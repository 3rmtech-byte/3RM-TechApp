# Documentación del Proyecto

## Características
- Aplicación móvil desarrollada con MIT App Inventor
- Funcionalidades principales incluyen autenticación de usuario, registro, y acceso a datos en tiempo real.

## Estructura del Proyecto
- **/ app**: contiene todos los componentes de la aplicación.
- **/ assets**: contiene recursos como imágenes y archivos adicionales.

## Configuración de MIT App Inventor
Para configurar el proyecto en MIT App Inventor:
1. Inicia sesión en MIT App Inventor.
2. Importa el archivo .aia del proyecto.
3. Conecta tu dispositivo o emulador.

## Guía para Comenzar
1. Clona el repositorio: `git clone https://github.com/3rmtech-byte/3RM-TechApp`
2. Navega a la carpeta del proyecto:
```
cd 3RM-TechApp
```
3. Abre el proyecto en MIT App Inventor.
4. Sigue la configuración mencionada anteriormente.
